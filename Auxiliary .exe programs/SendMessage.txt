Send a System-Defined Message to a given or current window.

SendMessage ["title"] Message [wParam [lParam]]

    "title"      Title of the recipient window.
    Message      Number of the message to send.
    wParam       First parameter for the message.
    lParam       Second parameter for the message.

A detailed description of this program is given here:
http://www.dostips.com/forum/viewtopic.php?f=3&t=7583

In the SendMessageValues.bat companion file there are definitions for all
message numbers that have been successfully tested in a cmd.exe window.
