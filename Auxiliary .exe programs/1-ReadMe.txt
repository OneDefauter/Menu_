Nesta pasta eu carreguei os programas .exe do Windows que escrevi como comandos auxiliares
para arquivos Batch, mais alguns arquivos Batch de suporte; cada programa .exe tem um .txt
arquivo acompanhante com a descrição do programa. Esta é uma lista dos programas
incluído neste site, agrupado por tipo de operação.


Manipulação de cordas:

StrLen.exe Obtenha o comprimento das literais das cordas e das variáveis Batch.
Ascii.exe Mostrar os códigos Ascii das literais de string e das variáveis de Batch.

Teclado de entrada:

GetKey.exe Obter uma chave do teclado e retornar seu valor.
ShowKeyCodes.bat Mostrar os códigos retornados por GetKey para todas as teclas.
FlushInputBuffer.exe Limpar o buffer de entrada de qualquer pressionamento de tecla anterior.
GetInput.exe Ler a entrada de ambas as teclas do teclado e os cliques do mouse.

Saída da tela:

CursorPos.exe Obter ou definir a posição do cursor.
CursorSize.exe Obter ou ajustar o tamanho do cursor.
Show.exe Mostrar caracteres de códigos, literais e variáveis Ascii.
ColorShow.exe Mostrar caracteres da mesma forma que Show.exe, mas em cores.
ColorChar.exe Versão despojada do ColorShow.exe, sem variáveis.
ColorBox.exe Mostrar uma caixa em modo texto usando caracteres de traçado de linha.
DefineColorCodes.bat Define valores úteis para escrever atributos de cor.
Window.exe Obter/definir vários valores relacionados à janela da tela do console.

Data e hora:

StdTime.exe Obter uma hora padrão e convertê-la para milissegundos.
ElapsedTime.bat Exibe o tempo decorrido em um formato padrão.
StdDate.exe Obtenha uma data padrão e converta-a para Julian Day Number.
EasterSunday.bat Calcule a data do domingo de Páscoa cristã.

Gerenciamento de arquivos:

GetFileType.exe Obtenha o tipo do arquivo conectado a um manipulador de arquivos.
FilePointer.exe Mover o ponteiro do arquivo do arquivo de disco redirecionado.
TruncadoFile.exe Truncar o tamanho do arquivo de disco redirecionado.
ReadFile.exe Ler bytes de um manipulador de arquivo de entrada redirecionado.

Diversos programas:

printf.exe Mostrar a saída formatada e avaliar expressões aritméticas RPN.
SendMessage.exe Enviar uma mensagem definida pelo sistema para uma janela.

A maioria dos programas está totalmente descrita em:
http://www.dostips.com/forum/viewtopic.php?f=3&t=3428
Alguns programas incluem um link para exemplos particulares no arquivo .txt companheiro
Você pode revisar e postar perguntas sobre esses programas nos links indicados.

In this folder I uploaded the Windows .exe programs I wrote as auxiliary commands
for Batch files, plus some support Batch files; each .exe program have a .txt
companion file with the program description. This is a list of the programs
included on this site grouped by type of operation.


String manipulation:

StrLen.exe              Get the length of string literals and Batch variables.
Ascii.exe               Show Ascii codes of string literals and Batch variables.

Keyboard input:

GetKey.exe              Get a key from keyboard and return its value.
ShowKeyCodes.bat        Display the codes returned by GetKey for all keys.
FlushInputBuffer.exe    Clear the input buffer from any previous keystroke.
GetInput.exe            Read input from both keyboard keys and mouse clicks.

Screen output:

CursorPos.exe           Get or set cursor position.
CursorSize.exe          Get or set cursor size.
Show.exe                Show characters from Ascii codes, literals and variables.
ColorShow.exe           Show characters in the same way as Show.exe, but in color.
ColorChar.exe           Stripped down version of ColorShow.exe with no variables.
ColorBox.exe            Show a box in text mode using line-drawing characters.
DefineColorCodes.bat    Define values useful to write color attributes.
Window.exe              Get/set several values related to console screen window.

Date and Time:

StdTime.exe             Get a standard time and convert it to milliseconds.
ElapsedTime.bat         Display elapsed times in a standard format.
StdDate.exe             Get a standard date and convert it to Julian Day Number.
EasterSunday.bat        Calculate the date of Christian Easter Sunday.

File handle management:

GetFileType.exe         Get the type of the file connected to a file handle.
FilePointer.exe         Move the file pointer of the redirected disk file.
TruncateFile.exe        Truncate the size of the redirected disk file.
ReadFile.exe            Read bytes from a redirected input file handle.

Miscellaneous programs:

printf.exe              Show formatted output and evaluate RPN arithmetic expressions.
SendMessage.exe         Send a System-Defined Message to a window.

Most programs are fully described at:
http://www.dostips.com/forum/viewtopic.php?f=3&t=3428
Certain programs include a link to particular examples in the companion .txt
file; you may review and post questions about these programs at the given links.
